  <ul>

  <h2>Installation ⚙️</h2>
  <ol>

    <li>Navigate to the <code>custom-dialog-page</code> directory:</li>
    <code>cd custom-dialog-page</code>
    <li>Install the required dependencies:</li>
    <code>flutter pub get</code>
